Istruzioni per l'utente 
Per avviare il programma e gestire le proprie spese, seguire i semplici passaggi riportati di seguito: 
1. Assicurarsi di avere installato l'interprete Python (versione 3.x). 
2. Posizionarsi all'interno della cartella del progetto contenente il file Elaborato.py. 
3. Per eseguire il programma, è possibile procedere in due modi: 
o Tramite terminale: Aprire il prompt dei comandi, navigare nella cartella del progetto e 
digitare il comando python main.py. 
o Tramite IDLE: Aprire il file main.py con l'interfaccia di IDLE e premere il tasto F5 sulla 
tastiera. 
4. Al primo avvio, il sistema creerà automaticamente il database spese_personali.db nella 
cartella di lavoro; non è necessaria alcuna configurazione manuale preliminare.
